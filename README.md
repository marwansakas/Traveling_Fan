# Traveling Fan Problem

This repository contains an implementation of the Traveling Fan Problem using various algorithms and techniques. It includes modules for genetic algorithms, utility functions, and more.

## Installation

To install the necessary dependencies, please run:

```
pip install -r requirements.txt
```

## Usage

To run the main program, execute:

```
python main.py
```

## Files Included

- **genetic_algo.py**: Contains the genetic algorithm implementation.
- **main.py**: The main script to run the application.
- **manager.py**: Manages the flow of the application.
- **output.txt**: Output file for results.
- **requirements.txt**: Python dependencies required for the project.
- **stadium.py**: Contains the stadium model or functions.
- **utils.py**: Utility functions used throughout the project.
- **UI/**: User interface elements.
- **data/**: Data inputs and outputs.
- **__pycache__/**: Compiled Python files.
